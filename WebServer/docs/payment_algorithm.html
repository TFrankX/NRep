Алгоритм работы с платежами (Stripe)

Система: ShareCharging - аренда powerbank'ов
Дата: Март 2026

1. Пользователь берёт powerbank

Пользователь сканирует QR → Do(deviceId) в UserController

Если device.TypeOfUse — один из PayByCard вариантов:

  1. Резервируется powerbank — pbTake.Reserved = true
  2. Создаётся Stripe Checkout Session с capture_method: "manual" (hold): 
    Amount = pricingPlan.HoldAmount  (например, 25 EUR)
  3. Пользователь вводит карту на странице Stripe
  4. После успешной оплаты → redirect на AlreadyPaid
  5. Деньги заблокированы на карте (не списаны!)
  6. Powerbank выдаётся, pbPush.SessionId = session_Id

2. Пока powerbank у пользователя

Фоновый сервис ScanDevices периодически проверяет:

if (powbank.SessionId != "" && powbank.SessionId != "\"\"")
{
    // Рассчитываем текущую стоимость
    powbank.Cost = _pricingService.CalculateCost(
        typeOfUse, powbank.LastGetTime, powbank.Taken);

    // Если прошло MaxDaysBeforeCapture дней — списываем ВСЮ сумму
    if ((DateTime.Now - powbank.LastGetTime).Days >= pricingPlan.MaxDaysBeforeCapture)
    {
        stripeRoutines.MakePostPayment(pricingPlan.HoldAmount);
        powbank.SessionId = "";  // Больше не отслеживаем
    }
}
Важно: Если пользователь не вернул powerbank в течение MaxDaysBeforeCapture дней, система автоматически списывает полную сумму hold (считаем powerbank потерянным).

3. Пользователь возвращает powerbank

Когда powerbank вставлен обратно в станцию:

pb.Cost = _pricingService.CalculateCost(
    device.TypeOfUse, pb.LastGetTime, pb.Taken);

if (pb.Cost > 0)
{
    // Списываем только фактическую стоимость
    stripeRoutines.MakePostPayment(pb.Cost);  // Capture частичной суммы
}
else
{
    // Если вернули за < 1 минуту — отменяем hold
    stripeRoutines.ReleaseHeldPayment();  // Cancel PaymentIntent
}
pb.SessionId = "";

Расчёт стоимости (PricingService.CalculateCost)

Формула:

Если < 1 минуты → 0 EUR (бесплатно)

Первый день:
cost = BaseFee + ceil(hours) × HourlyRate

Со второго дня:
cost = (BaseFee + 24 × HourlyRate) + days × DailyRate

Максимум: HoldAmount (cap)

Пример для PayByCard (стандартный план)

Время аренды Расчёт Стоимость
30 секунд бесплатно 0 EUR
1 час 2 + 1×2 4 EUR
3 часа 2 + 3×2 8 EUR
12 часов 2 + 12×2 = 26 25 EUR (cap)
2 дня (2 + 24×2) + 1×12 = 62 25 EUR (cap)

Схема денежного потока

БЕРЁТ POWERBANK

Карта → Stripe Hold 25 EUR
(деньги заморожены на карте, но НЕ списаны)

↓

ВРЕМЯ АРЕНДЫ ИДЁТ

Если прошло ≥ 3 дней (MaxDaysBeforeCapture):
→ Capture 25 EUR (полное списание, считаем powerbank потерянным)

↓

ВОЗВРАЩАЕТ POWERBANK

Вариант A: < 1 минуты
→ Cancel hold (0 EUR списано)

Вариант B: 1 мин - 3 дня
→ Capture фактической стоимости (например, 6 EUR)
→ Остаток (19 EUR) автоматически размораживается
